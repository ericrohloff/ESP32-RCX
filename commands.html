Python API (what students use)

This page documents the Python wrapper functions provided by the ESP32 standalone driver (the `rcx` object in esp32_driver.py). High-school students can use these functions directly in their MicroPython scripts. The low-level LASM-to-bytes conversion is done internally and is intentionally hidden.

Available functions on the `rcx` object

  • beep() - Play a short beep using the IR transmitter. 
    rcx.beep()
  • motor_on(m) - Turn motor number m on. Valid motors: 0, 1, 2. 
    rcx.motor_on(0)  # turn motor 0 on
  • motor_off(m) - Turn motor number m off. 
    rcx.motor_off(0)  # turn motor 0 off
  • stop_all_tasks() - Send a "stop" command to the RCX to halt running tasks. 
    rcx.stop_all_tasks()
  • delete_task(tid) - Delete a stored task definition from the interpreter's local store. (This does not directly map to hardware; it's for the script VM.) 
    rcx.delete_task(0)
  • set_log_level(n) - Set the interpreter's local log level (for debugging only). 
    rcx.set_log_level(1)
  • tmrz(n) - Reset software timer number n to zero (internal VM timer). 
    rcx.tmrz(2)
  • wait(a, b) - Pause execution. For compatibility with older scripts the driver treats the wait duration as b * 0.1 seconds. 
    rcx.wait(2, 5)  # ~0.5 seconds
  • set_var(idx, val) - Store a numeric value in the interpreter variable table. Use this to hold counters, parameters, or references. 
    rcx.set_var(5, 42)
  • mulv(dst, src, mul) - Multiply variable src by mul and store the low 8 bits in dst (useful for simple arithmetic). 
    rcx.mulv(10, 2, 20)
  • out_port(port, value) - Drive outputs according to the bits of value. By default bit0→motor0, bit1→motor1, bit2→motor2. 
    rcx.out_port(2, 5)  # set port 2 to value 5 (bits 0 and 2 set)

Useful VM state

  • rcx.vars - dictionary of interpreter variables (read/write).
  • rcx.tasks - stored task definitions created by the interpreter (if used).

Quick example script

# Example: simple motor pulse sequence
rcx.motor_on(0)
rcx.wait(0, 20)   # ~2.0 seconds using the default scaling
rcx.motor_off(0)
rcx.beep()

Quick Actions

Use the app to compose scripts and flash them to the ESP32. The interpreter will run the stored commands when executed on-device.
